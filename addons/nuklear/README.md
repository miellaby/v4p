# Nuklear Addon for V4P

This addon integrates the Nuklear immediate-mode GUI library with V4P. It provides a way to create user interfaces directly within V4P applications.

## Status

**Note**: This addon is currently not functional. It is a work in progress and not yet ready for use.

## Overview

Nuklear is a lightweight, immediate-mode GUI library designed for embedded systems and applications. This addon aims to bridge Nuklear with V4P, allowing developers to create interactive UIs within their V4P-based applications.

## Files

- `nuklear.h`: Copy of the Nuklear library header file.
- `nuklear_v4p.h`: Header file for V4P Nuklear V4P/G4P implementation.

## Usage

Once functional, this addon will allow you to:

- Create windows, buttons, sliders, and other UI elements.
- Render UI elements via V4P.
- Handle user input and interactions with G4P.
- See demos/demo_nuklear.c
